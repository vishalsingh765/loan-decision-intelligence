# 💳 Loan Decision Intelligence

An AI-powered web application that predicts whether a loan application will be approved or rejected based on a user's financial and personal information. The system leverages a machine learning model and provides real-time risk assessment through an interactive, modern user interface.

## 🚀 Features

* Real-time loan approval prediction
* Modern and responsive user interface
* Machine learning–based decision engine
* Credit score and financial profile analysis
* Dynamic approval/rejection visualization
* REST API integration with backend ML model
* Input validation and error handling
* Mobile-friendly design

## 🛠️ Tech Stack

## 📋 Input Parameters

The model evaluates loan eligibility using:

* Age
* Annual Income
* Loan Amount
* Credit Score
* Employment Experience
* Education Level
* Housing Status

## 🎯 How It Works

1. Users enter their financial information.
2. The frontend validates the inputs.
3. Data is sent to the Flask API.
4. The machine learning model processes the request.
5. The system returns an approval or rejection prediction with additional insights.
